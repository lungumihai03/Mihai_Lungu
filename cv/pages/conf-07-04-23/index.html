Certificat

Valorificarea competențelor de programare la elaborarea unei aplicații de gestionare a tablourilor de structuri

CZU: 004.4

Pag. 166-172

Lungu Mihai, Pavel Maria

Universitatea Pedagogică de Stat „Ion Creangă“ din Chişinău

Rezumat:

În lucrare se expune o metodologie de valorificare a competențelor de programare a studenților anului I, ciclul I, specialitatea Informatică, la studierea structurilor în limbajul de programare C. Pentru realizarea lucrului individual, studenții cu performanțe înalte de programare, sunt motivați în utilizarea platformelor de programare vizuală, care facilitează învățarea tipurilor structurate de date, precum structurile (înregistrările, articolele) și fișierele. Metodologia utilizată este exemplificată la elaborarea aplicației de gestionare a unui depozit de automobile.

The paper presents a methodology for leveraging the programming skills of first-year students, first cycle, Computer Science major, when studying structures in the C programming language. To achieve individual work, students with high programming performance are motivated to use the visual programming platforms, which facilitates learning structured data types such as structures (records, items) and files. The used methodology is exemplified when developing the application for managing a car warehouse.
