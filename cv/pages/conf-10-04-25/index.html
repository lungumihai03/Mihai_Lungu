Dezvoltarea și implementarea unui modul software pentru gestionarea activităților în domeniul resurselor umane și al contabilității

Pag. 530-538

Lungu Mihai, Gasnaş Ala

Universitatea Pedagogică de Stat „Ion Creangă“ din Chişinău

Rezumat:

This paper presents the development of a desktop application designed to streamline human resources and payroll management for small and medium-sized enterprises. Built using modern technologies such as WinUI 3, the C# programming language, and Visual Studio 2022, the application provides an intuitive graphical interface and essential functionalities. For recruiters, it offers tools for adding employees, viewing records, and managing leaves, while for accountants, it automates salary calculations, generates payroll slips in Excel format, and creates detailed reports. The MySQL 10.4.28 database, connected via MySQL Connector, ensures efficient data storage for employee, department, and salary information, supporting complex queries. The development followed object-oriented programming principles, resulting in a modular and scalable architecture. Key outcomes include automated processes, quick data access, enhanced security, and flexibility for future enhancements. The project highlights the importance of selecting appropriate technologies and rigorous design, delivering a valuable tool for optimizing administrative tasks

Cuvinte cheie:

Human resources, payroll, Desktop application, MySQL, Object-orientedprogramming
